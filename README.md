# Favourite Movies

A base application to start a Modern Rails Application

## Setup

```sh
git clone git@github.com:renuo/favourite-movies.git
cd favourite-movies
bin/setup
```

### Configuration

No configuration needed

### Run

```sh
bin/run
```

### Tests / Checks

```sh
bin/check
```
